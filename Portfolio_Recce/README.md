# Drone Operator Portfolio

This folder contains a complete static portfolio website for a drone operator.

## Quick Start

1. Open `index.html` in your browser.
2. Replace image and video placeholders inside `assets/images` and `assets/videos`.
3. Update text, contact links, and drone specs in `index.html`.

## Media Placeholders Used

- Hero video: `assets/videos/hero-reel.mp4`
- Hero poster image: `assets/images/hero-poster.jpg`
- Work gallery images:
  - `assets/images/work-1.jpg`
  - `assets/images/work-2.jpg`
  - `assets/images/work-3.jpg`
  - `assets/images/work-4.jpg`

## Drone Specs Section

The section "Two Platforms, Two Mission Styles" in `index.html` contains 2 cards:

- Drone Type A
- Drone Type B

Replace each card's details with your real drone model names and exact specs.

## Deploy Options

You can deploy this portfolio quickly on:

- Netlify (drag-and-drop folder)
- Vercel (import project)
- GitHub Pages
